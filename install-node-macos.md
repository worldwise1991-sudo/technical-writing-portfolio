# How to Install Node.js on macOS

## Introduction
This guide explains how to install Node.js on macOS. It is written for beginners and does not require any prior experience.

By the end of this guide, you will have Node.js installed and be able to verify that it is working correctly.

## Step 1: Download Node.js
Go to https://nodejs.org/.

Click the button labeled **LTS (Recommended for most users)** to download the installer.

## Step 2: Install Node.js
Open the downloaded '.pkg' file.

Follow the installation prompts by clicking **Continue** and then **Install**.

You may be asked to enter your Mac password to complete the installation.

## Step 3: Verify Installation
Open Terminal and run the following command:

'''bash
node -v

If Node.js was installed successfully, you will see a version number (for example, v24.15.0).

If you see an error instead, try restarting Terminal and running the command again.

## Conclusion

You have successfully installed Node.js on macOS.

You can now begin using Node.js to run JavaScript applications or continue learning more about development tools.
